## 配置命令行

Linux 图形化系统发展相对缓慢，尤其是很多底层配置设定 GUI 界面都没有适配好、甚至完全没有，
因此日常 Linux 使用还是要依赖命令行控制，更何况 Linux 命令行拥有大批量的社区技术爱好者维护改善，

## 软件推荐

- [pipx](https://github.com/pipxproject/pipx)

  可避免 pip 包依赖冲突的 python 命令行工具安装器

  ```bash
  # 安装
  python3 -m pip install --user pipx

  # 加入环境变量的指导
  python3 -m pipx ensurepath

  # 激活自动补全功能的指导
  pipx completions
  ```

&! 可脱离终端启动 GUI 程序

以下情况建议 `Ctrl` + `Alt` + `F2` ~ `F6` 启动一个字符流界面的 tty 进行操作：

- 会影响桌面系统的系统包库升级
- 系统桌面崩溃了
