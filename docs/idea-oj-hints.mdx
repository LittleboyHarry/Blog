## 调式快捷键

- <MyIcons.breakpoint /> 设置/取消行断点 <code>Ctrl + F8</code>
- <MyIcons.debug /> 开始/重新调试 <code>Shift + F9</code>

## 输入重定向

![](/figures/idea-oj-hints/redirect-input.png)

在文本框填入源文本文件的项目相对路径（其他亦可）

## UX 理念

- 在对话框中，<button className="button button--sm button--primary">着色的按钮</button> 代表主要按钮的意思，
  按下 `Enter` 键或 `Space` 键立刻触发，`Esc` 键有关闭 Cancel 的语义
